ה-outcomes-grader כולל עכשיו את שכבת ה-Code Review 6-stage (FIRST PASS → FAIL TRIGGERS → EDGE CASES → BEHAVIORAL DIFF → SAFETY AUDIT → VERDICT) + Anti-Premature Closure check, שהיו פעם בסוכן הנפרד `code-reviewer`.
