# ניווט — תהליך עבודה

**שינוי 2026-05-26:** הסוכן `navigator-process-guide` נמחק כי הניווט הוא משימה של ה-Lead Agent (Claude Code הראשי) ולא דורש חלון context נפרד.

הפקודה הזו מבקשת מה-Lead Agent לעצור ולסכם:

## מה Lead Agent מציג:
```
📍 איפה אתה עכשיו:
   Stage: [Intent / Investigation / Checkpoint / Planning / Code / Gates]
   App: [User App / Admin Panel / Functions / Shared]
   Env: [DEV / PROD]
   Branch: [current-branch-name]

✅ מה כבר נעשה:
   - [רשימת שלבים שהושלמו]

➡️ הצעד הבא המומלץ:
   /[command] [args]
   או
   Agent: [agent-name]
   הסבר: [למה זה הצעד הנכון]

⚠️ Checkpoints שעדיין חסרים:
   - [אם יש]
```

## מפת הפרוטוקול (מ-CLAUDE.md):
1. **Intent** — Haim מגדיר מה רוצים (עם `/intent` אם הבקשה מעורפלת)
2. **Investigation** — מיפוי הבעיה (workers במקביל) — READ ONLY
3. **Checkpoint** — Haim מאשר או שולל (עם findings מ-completeness-checker)
4. **Planning** — תכנון עם אופציות (Lead Agent)
5. **Code** — הקוד (Lead Agent + workers)
6. **Gates** — `outcomes-grader` (מאחד code review + PROD safety)

**חובה לעבור בסדר. אסור לדלג על Checkpoint.**
