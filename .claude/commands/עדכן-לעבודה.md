# סנכרון - דחיפה ל-GitHub

בצע את הפעולות הבאות **בדיוק לפי הסדר**, ועצור מיד אם יש שגיאה:

1. הרץ: `git status` - הצג את המצב הנוכחי
   - **אם יש שגיאה - עצור והצג את השגיאה למשתמש. אל תמשיך.**

2. הרץ: `git add .` - הוסף את כל השינויים
   - **אם יש שגיאה - עצור והצג את השגיאה למשתמש. אל תמשיך.**

3. הרץ: `git commit -m "עדכון - <תאריך ושעה נוכחיים בפורמט: YYYY-MM-DD HH:MM>"` - צור commit
   - **אם יש שגיאה - עצור והצג את השגיאה למשתמש. אל תמשיך.**

4. הרץ: `git push origin main` - דחוף ל-GitHub
   - **אם יש שגיאה - עצור והצג את השגיאה למשתמש.**

✅ **רק אם הכל עבר בהצלחה** - אשר למשתמש: "הפרויקט עודכן בהצלחה ל-GitHub. אתה יכול למשוך אותו במחשב השני."

⛔ **חשוב:**
- אל תנסה לפתור שגיאות לבד
- אל תריץ פקודות נוספות מעבר למה שרשום
- אם משהו לא עובד - דווח למשתמש ועצור
