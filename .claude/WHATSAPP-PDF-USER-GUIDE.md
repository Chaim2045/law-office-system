# 📎 מדריך משתמש - העלאת הסכמי שכ"ט דרך WhatsApp

## 🎯 מה זה?

תכונה חדשה המאפשרת לך להעלות הסכמי שכ"ט (או כל מסמך PDF/תמונה) ישירות דרך WhatsApp ולשייך אותם ללקוח במערכת.

---

## 🚀 איך משתמשים?

### שלב 1: שלח את המסמך ב-WhatsApp

1. פתח את שיחת ה-WhatsApp עם הבוט
2. לחץ על סמל הקליפס 📎 או המצלמה 📷
3. בחר את הקובץ (PDF או תמונה)
4. **חשוב:** בשדה הכיתוב (Caption) כתוב את שם הלקוח
   - דוגמה: `דוד כהן`
   - דוגמה: `רחל לוי`
5. שלח את ההודעה

### שלב 2: בחר את הלקוח הנכון

הבוט יחפש לקוחות מתאימים ויציג לך רשימה:

```
📎 מסמך התקבל!

━━━━━━━━━━━━━━━━━━━━
📄 סוג: PDF
💾 גודל: 245KB
━━━━━━━━━━━━━━━━━━━━

🔍 נמצאו 2 לקוחות מתאימים:

1️⃣ דוד כהן
   ת.ז. 123456789
   📞 052-1234567

2️⃣ דוד כהן (בן דוד)
   ת.ז. 987654321
   📞 054-9876543

━━━━━━━━━━━━━━━━━━━━
💡 כתוב מספר לאישור (1-2)
❌ או כתוב "ביטול" לביטול
```

כתוב את המספר של הלקוח הנכון (למשל: `1`)

### שלב 3: קבל אישור

הבוט יעלה את המסמך ויאשר:

```
✅ הסכם שכ"ט הועלה בהצלחה!

━━━━━━━━━━━━━━━━━━━━
👤 לקוח: דוד כהן
🆔 ת.ז. 123456789
📄 קובץ: agreement_1702393847123.pdf
💾 גודל: 245KB
👤 הועלה על ידי: שם שלך
━━━━━━━━━━━━━━━━━━━━

המסמך נשמר בכרטיס הלקוח ויופיע באדמין פאנל.

כתוב "תפריט" לחזרה לתפריט ראשי
```

**זהו! המסמך כעת מופיע בכרטיס הלקוח באדמין פאנל.**

## ✅ טיפים לשימוש נכון

### 1. **שם הלקוח בכיתוב**
- **חובה** לכתוב שם לקוח בכיתוב
- השתמש בשם פרטי או משפחה או שניהם
- דוגמאות טובות:
  - ✅ `דוד כהן`
  - ✅ `דוד`
  - ✅ `כהן`
- דוגמאות רעות:
  - ❌ *(ריק - ללא כיתוב)*
  - ❌ `הסכם חדש` *(לא שם לקוח)*

### 2. **סוגי קבצים נתמכים**
- ✅ PDF (מומלץ)
- ✅ תמונות (JPEG, PNG, WebP)
- ❌ Word, Excel, או קבצים אחרים

### 3. **גודל קובץ**
- **מקסימום: 10MB**
- אם הקובץ גדול מדי, נסה:
  - לדחוס את ה-PDF
  - לקטין את איכות התמונה
  - לפצל לכמה קבצים

### 4. **בדיקה אחרי העלאה**
- היכנס לאדמין פאנל
- פתח את כרטיס הלקוח
- בדוק ש-המסמך מופיע בסעיף "הסכמי שכ"ט"
- אפשר להוריד ולוודא שזה הקובץ הנכון

## 🔍 חיפוש לקוחות - איך זה עובד?

הבוט משתמש בחיפוש **חכם ומדויק**:

### 1. **התאמה מדויקת** (עדיפות ראשונה)
- אם כתבת `דוד כהן` והיא בדיוק שם של לקוח - זה יופיע ראשון

### 2. **מתחיל ב...** (עדיפות שנייה)
- אם כתבת `דוד` והיש לקוח ששמו מתחיל ב-`דוד...` - זה יופיע שני

### 3. **מכיל את המילה** (עדיפות שלישית)
- אם כתבת `כהן` והיש לקוח ששמו `דוד כהן` - זה יופיע

### 4. **מילים נפרדות**
- אם כתבת `דוד` והיש `דוד לוי` ו-`יוסי בן דוד` - שניהם יופיעו

### דוגמאות:
| מה שכתבת | מי יופיע | מי לא יופיע |
|----------|----------|--------------|
| `דוד` | דוד כהן, דוד לוי, יוסי בן דוד | רחל, משה |
| `כהן` | דוד כהן, רחל כהן | דוד לוי |
| `דוד כהן` | דוד כהן (מדויק ראשון), דוד כהן בן דוד | רחל כהן |

## ❌ שגיאות נפוצות ופתרונות

### 1. "לא נמצא לקוח בשם..."
**בעיה:** הבוט לא מצא לקוח מתאים

**פתרונות:**
- בדוק איות (אולי יש טעות?)
- נסה רק שם פרטי או רק משפחה
- בדוק שהלקוח קיים במערכת
- בדוק שהלקוח **פעיל** (לא מושבת)

### 2. "הקובץ גדול מדי"
**בעיה:** הקובץ מעל 10MB

**פתרונות:**
- השתמש בכלי דחיסת PDF (PDF Compressor)
- העלה תמונה במקום PDF (אם זה scan)
- קטן את איכות התמונה
- פצל לכמה קבצים

### 3. "סוג קובץ לא נתמך"
**בעיה:** שלחת Word, Excel, או סוג קובץ אחר

**פתרון:**
- המר ל-PDF
- או צלם תמונה של המסמך

### 4. "יש לך כבר מסמך הממתין לאישור"
**בעיה:** שלחת מסמך נוסף לפני שסיימת עם הקודם

**פתרון:**
- סיים את התהליך הנוכחי (בחר מספר או כתוב "ביטול")
- אז שלח את המסמך הבא

### 5. "שגיאה בהעלאת המסמך"
**בעיה:** בעיה טכנית בהעלאה

**פתרונות:**
- נסה שוב
- בדוק חיבור אינטרנט
- אם זה לא עובד - דווח למנהל המערכת
