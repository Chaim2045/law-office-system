🎉 מערכת הודעות חדשה

מערכת הודעות מודרנית, מודולרית ומעוצבת למשרד עורכי הדין

✅ הודעות הצלחה (Success)

הודעות ירוקות למשתמש כשפעולה הושלמה בהצלחה

// דוגמה לשימוש NotificationSystem.success('הפעולה הושלמה בהצלחה'); NotificationSystem.show('משימה נוספה', 'success', 3000);

❌ הודעות שגיאה (Error)

הודעות אדומות למשתמש כשיש בעיה או שגיאה

// דוגמה לשימוש NotificationSystem.error('שגיאה בשמירת הנתונים'); NotificationSystem.show('המשימה לא נמצאה', 'error', 0); // 0 = ללא סגירה אוטומטית

⚠️ הודעות אזהרה (Warning)

הודעות כתומות לאזהרות ולמידע חשוב

// דוגמה לשימוש NotificationSystem.warning('נגמרה יתרת השעות ללקוח'); NotificationSystem.show('שים לב! יש 5 משימות דחופות', 'warning');

ℹ️ הודעות מידע (Info)

הודעות כחולות למידע כללי ועדכונים

// דוגמה לשימוש NotificationSystem.info('הנתונים עודכנו בהצלחה'); NotificationSystem.show('יש 3 הודעות חדשות', 'info', 5000);

⏳ מסך טעינה (Loading)

מסך טעינה מלא עם ספינר מעוצב

// דוגמה לשימוש NotificationSystem.showLoading('שומר נתונים...'); // ... ביצוע פעולה ... NotificationSystem.hideLoading();

❓ דיאלוג אישור (Confirm)

דיאלוג מעוצב לאישור פעולות קריטיות

// דוגמה לשימוש NotificationSystem.confirm( 'האם אתה בטוח שברצונך למחוק את המשימה?', () => { console.log('אושר!'); NotificationSystem.success('המשימה נמחקה'); }, () => { console.log('בוטל'); }, { title: 'אישור מחיקה', confirmText: 'מחק', cancelText: 'ביטול', type: 'error' } );

🌟 תכונות המערכת

  • 4 סוגי הודעות: Success, Error, Warning, Info
  • עיצוב מודרני עם אנימציות חלקות
  • תמיכה ב-Stack של עד 3 הודעות בו-זמנית
  • סגירה אוטומטית עם אפשרות להגדיר זמן
  • כפתור X לסגירה ידנית
  • מסך טעינה מלא עם ספינר מעוצב
  • דיאלוג אישור מעוצב (מחליף את confirm)
  • תמיכה מלאה ב-RTL (עברית)
  • Responsive לכל המסכים
  • תמיכה מלאה ב-Accessibility
  • הגנה מפני XSS
  • מודולרי - קל להוסיף ולשנות

📖 איך להשתמש?

שלב 1: הוסף את הקבצים ל-HTML שלך:

<!-- CSS --> <link rel="stylesheet" href="notifications.css"> <!-- JavaScript --> <script type="module"> import NotificationSystem from './js/modules/notification-system.js'; window.NotificationSystem = NotificationSystem; </script>

שלב 2: השתמש בקוד:

// הודעות פשוטות NotificationSystem.success('הפעולה בוצעה בהצלחה'); NotificationSystem.error('אירעה שגיאה'); NotificationSystem.warning('שים לב!'); NotificationSystem.info('מידע חשוב'); // טעינה NotificationSystem.showLoading('טוען...'); NotificationSystem.hideLoading(); // אישור NotificationSystem.confirm( 'האם אתה בטוח?', () => console.log('כן'), () => console.log('לא') );
