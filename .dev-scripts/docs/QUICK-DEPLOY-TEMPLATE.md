# ⚡ תבנית קצרה לפריסה - להעתקה מהירה

## 🎯 העתק והדבק את זה בסוף כל צ'אט:

```
קלוד, סיימנו עם העבודה.

⚠️ חשוב: אני עובד עם כמה צ'אטים של קלוד במקביל!

תהליך פריסה מלא:

שלב 0 - בדיקת קומיטים לא מוכרים:
──────────────────────────────────────
• git log --oneline -10
• יש קומיטים שאתה לא זוכר שעשית?
• אם כן: git show [hash] --stat
• בדוק 3 סימנים:
  ✓ Author: Chaim2045 (השם שלי)
  ✓ "Generated with Claude Code"
  ✓ "Co-Authored-By: Claude"
• אם כל 3 קיימים = קלוד אחר שלי (בסדר!)
• אם לא = תזהיר מיד ואל תמשיך!

שלב 1 - ניתוח קבצים:
──────────────────────────────────────
• git status && git diff --name-only
• סווג כל קובץ:
  - master-admin-panel/ → אדמין
  - מחוץ לתיקייה → ממשק משתמשים

שלב 2 - דיווח מפורט:
──────────────────────────────────────
דווח לי:
✓ קומיטים לא מוכרים? (כן/לא + פרטים)
✓ קבצים ששונו (רשימה מלאה)
✓ סיווג: אדמין / ממשק / שניהם
✓ יעד פריסה:
  • אדמין → https://admin-gh-law-office-system.netlify.app
  • ממשק → https://gh-law-office-system.netlify.app
  • שניהם → שני האתרים
✓ Branch נוכחי: main או feature/?
✓ קבצים זרים שלא קשורים לעבודה שלנו?

שלב 3 - המתן לאישור:
──────────────────────────────────────
חכה לאישור מפורש ממני לפני המשך!

שלב 4 - ביצוע (רק אחרי אישור):
──────────────────────────────────────
• git add [הקבצים שלי בלבד]
• git commit -m "[הודעה מקצועית]"
• git push origin [branch]

שלב 5 - סיכום:
──────────────────────────────────────
דווח:
✓ קומיט חדש שנוצר
✓ קומיטים קודמים שנדחפו (אם יש)
✓ לאן נדחף
✓ יתפרס ל: [URLs]
✓ זמן משוער: 2-3 דקות

עכשיו התחל משלב 0.
```

## 💡 גרסה עוד יותר קצרה (מינימלית):

```
קלוד, סיימנו.

לפני push דווח:
• קומיטים לא מוכרים?
• מה השתנה?
• אדמין או ממשק?
• איזה branch?

אחרי דיווח חכה לאישור.
```

## 🚀 גרסה אולטרה-מינימלית (1 שורה!):

```
קלוד, סיימנו. בדוק קומיטים+קבצים, דווח (קומיטים זרים? אדמין/ממשק? branch?), חכה לאישור, push.
```
